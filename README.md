# Sales Data Analysis using SQL

## Objective
To analyze sales dataset and identify revenue trends, top performing products, and monthly performance.

## Dataset
- Total Rows: 700

## KPIs Analyzed
- Total Revenue
- Product-wise Revenue
- Top and Bottom Products
- Quantity Sold
- Monthly Revenue Trends

## Key Insights
- Top Revenue Product: Oil (148,500)
- Lowest Revenue Product: Eggs (18,470)
- Total Revenue Generated: 698,715
- Highest Revenue Month: March

## Tools Used
- SQL Server
